# 二分查找算法说明

本文对应源码文件 `src/search/search.cpp`，包含二分查找及其左右边界查找的实现。

## 1. 二分查找适用条件

二分查找的前提是数据必须具有单调性，最常见的场景是“升序有序数组”。

如果数组无序，二分查找无法正确判断下一步应该去左边还是右边，此时应先排序或改用其他查找方法。

## 2. 标准二分查找

### 原理
每次取当前区间的中间位置，与目标值比较：

- 如果中间值等于目标值，查找成功。
- 如果中间值小于目标值，目标只可能在右半部分。
- 如果中间值大于目标值，目标只可能在左半部分。

每比较一次，搜索区间都会缩小一半。

### 流程
1. 设定左右边界 `left` 和 `right`。
2. 计算中间下标 `mid`。
3. 比较 `nums[mid]` 和目标值 `target`。
4. 根据比较结果收缩左边界或右边界。
5. 直到找到目标，或者区间为空。

### 复杂度
- 时间复杂度：`O(log n)`
- 空间复杂度：`O(1)`

## 3. 左边界二分查找

### 作用
当数组中存在重复元素时，用来查找目标值第一次出现的位置。

### 核心思路
即使 `nums[mid] == target`，也不立即返回，而是继续向左侧收缩区间，直到无法再缩小为止。

### 适用场景
- 统计某个值的出现范围
- 查找第一个满足条件的位置
- 处理重复元素问题

## 4. 右边界二分查找

### 作用
当数组中存在重复元素时，用来查找目标值最后一次出现的位置。

### 核心思路
即使 `nums[mid] == target`，也继续向右收缩区间，最终得到最后一次出现的位置。

### 适用场景
- 查找最后一个满足条件的位置
- 求某个值在有序数组中的右端点

## 5. 边界处理要点

二分查找最容易出错的部分不是“比较”，而是“区间定义”和“循环结束条件”。

本实现中：
- 标准二分查找使用左闭右闭区间 `[left, right]`
- 左边界和右边界查找使用左闭右开区间 `[left, right)`

这样写的好处是边界收缩更统一，更容易推导最终结果。

## 6. 总结

二分查找的本质是利用有序性不断排除不可能的区间。

如果只判断目标是否存在，使用标准二分查找即可。

如果数组中存在重复元素，并且需要精确定位第一个或最后一个位置，则应使用左右边界版本。
