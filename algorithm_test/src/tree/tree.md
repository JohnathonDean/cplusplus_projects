# 二叉树基础算法说明

本文对应源码文件 `src/tree/tree.cpp`，包含二叉树中最常见的基础内容：

- 二叉树节点定义

- 中序遍历
- 后序遍历
- 层序遍历
- 最大深度
- 二叉搜索树查找
- 二叉搜索树合法性判断

## 1. 二叉树基础概念

二叉树的每个节点最多只有两个孩子：
- 左孩子 `left`
- 右孩子 `right`

这里使用如下结构表示节点：

```cpp
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
};
```

## 2. 深度优先遍历

深度优先遍历的核心思想是：
先沿一条分支尽量向下访问，再回溯处理其他分支。

### 前序遍历

顺序：
1. 访问根节点
2. 遍历左子树
3. 遍历右子树

记忆方式：`根 -> 左 -> 右`

### 中序遍历

顺序：
1. 遍历左子树
2. 访问根节点
3. 遍历右子树

记忆方式：`左 -> 根 -> 右`

对于二叉搜索树，中序遍历结果一定是升序。

### 后序遍历

顺序：
1. 遍历左子树
2. 遍历右子树
3. 访问根节点

记忆方式：`左 -> 右 -> 根`

后序遍历特别适合“先处理子问题，再处理当前节点”的题目。

## 3. 层序遍历

层序遍历按树的层级一层一层访问节点，通常借助队列实现。

### 流程
1. 根节点入队
2. 取出队头节点并访问
3. 将它的左孩子和右孩子依次入队
4. 重复直到队列为空

### 特点
- 访问顺序是从上到下、从左到右
- 很适合求树的层数、最短路径或分层输出

## 4. 最大深度

最大深度表示：
从根节点到最远叶子节点路径上的节点数。

### 递归思路
对于任意节点：

```text
最大深度 = max(左子树深度, 右子树深度) + 1
```

空树深度为 `0`。

## 5. 二叉搜索树 BST

二叉搜索树满足：
- 左子树所有节点值都小于根节点
- 右子树所有节点值都大于根节点
- 左右子树本身也必须是二叉搜索树

### 查找

因为有“左小右大”的性质，所以查找时可以像二分一样缩小范围：
- 目标值等于当前节点，直接返回
- 目标值更小，去左子树
- 目标值更大，去右子树

### 合法性判断

只比较当前节点和左右孩子是不够的。

正确做法是给每个节点维护一个允许取值范围：
- 左子树必须落在 `(lower, root->val)` 内
- 右子树必须落在 `(root->val, upper)` 内

这样才能保证整棵树都满足 BST 性质。

## 6. 总结

二叉树模块里最基础、最常见的能力通常就是这些：
- 深度优先遍历
- 层序遍历
- 深度计算
- BST 查找
- BST 合法性判断

如果后续继续扩展，比较自然的下一步是：
- 最近公共祖先 `LCA`
- 平衡二叉树判断
- 路径总和
- 二叉树反转
- 二叉树序列化与反序列化
